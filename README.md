# Insider Threat Detection System

This project detects suspicious employee behavior using **Machine Learning** and provides a **web interface** to view the results.

## Features

- Upload employee activity logs (CSV files)
- Detect **Normal vs Suspicious** behavior using AI
- View results in an interactive **web interface**
- Built with Python, Flask, and Scikit-learn

## Technologies Used

- Python 3.14
- Flask
- Pandas
- Scikit-learn
- HTML, CSS
